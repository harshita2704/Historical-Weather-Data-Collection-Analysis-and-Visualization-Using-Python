# Historical Weather Data Collection, Analysis and Visualization Using Python

## 📌 Project Overview

This project focuses on collecting historical weather data using the Open-Meteo API and analyzing it using Python. The collected data is processed, cleaned, and explored through Exploratory Data Analysis (EDA) and interactive visualizations to understand weather trends and climatic patterns.

The project demonstrates the complete data analysis workflow, from API integration and data preprocessing to visualization and interpretation of weather data.

---

## 🎯 Objectives

- Collect historical weather data using the Open-Meteo API.
- Process and clean real-world weather datasets.
- Perform Exploratory Data Analysis (EDA).
- Visualize weather trends using different plotting libraries.
- Interpret weather patterns through graphical analysis.

---

## 📊 Dataset Information

The weather data was collected using the **Open-Meteo Historical Weather API**.

The dataset includes weather parameters such as:

- Temperature
- Humidity
- Rainfall
- Wind Speed
- Atmospheric Pressure
- Cloud Cover
- Visibility
